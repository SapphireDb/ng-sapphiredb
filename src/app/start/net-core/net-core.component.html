Get started - Setup server

Realtime database server runs on an Asp.Net Core application. First step is to setup the server to use Realtime database.
You need to have a working Asp.Net Core application and a configured DbContext.

Installation

Install package

To install the package execute the following command in your package manager console

You can also install the extension using Nuget package manager. The project can be found here: https://www.nuget.org/packages/RealtimeDatabase/

Configure DbContext

You now have to change your DbContext to derive from RealtimeContext. Also make sure to adjust the constructor parameters.
