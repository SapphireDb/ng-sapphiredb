Get started - Setup angular client

ng-realtime-database is a client implementation for Angular. You need a working Angular application.

Installation

Install package

To install the package execute the following command in your console

You can find the project here: https://www.npmjs.com/package/ng-realtime-database

Import realtime database module

To use the realtime database you have to first setup the realtime database. Import the module in your app.module.ts
REALTIME_DATABASE_OPTIONS is optional. ng-realtime-database will automatically get the current url, port and protocol and call the correct endpoint.
ng-realtime-database will also figure out a working connection type for your client.

Access the database

To call function on the database you have to inject the realtime database in your components etc.

Done

Your realtime database client should now work.

Start using it by querying data
