What is it?

Realtime database is an open source library that enables you to easily create your own realtime application.

Build amazing applications with less effort and get the best results of your project.
Realtime database should serve as a self hosted alternative to firebase.

All the Features You Need
Realtime database includes:

Dead simple configuration

Realtime database is very easy to configure and to use. It is aimed to bring results and not to waste time you could better use for own features. You can easily extend an existing application using EntityFrameworkCore.

Blazing fast development

Because everything is ready to use, realtime database increases development speed a lot.

Modern technologies

Realtime database uses modern technologies like ServerSentEvents, Websockets, Entity Framework and RxJs.

Self hosted

Self host your application and keep the control of everything. No third party services are used.

API generation

Realtime database generates interfaces for all models you need, so you don't have to care about basic functions like insert, update and delete.

Authentication/Authorization included

Realtime database comes with lots of features to secure you application and data. It also comes with a login provider.

Database support

Because realtime database is an extension for Entity Framework Core it also supports all database systems supported by EF Core.

Actions

Realtime database adds support for actions. Actions are server methods that can be called through the communication pipeline. Also supports async methods.

NLB support

Realtime database also supports NLB-scenarios and is scalable.
